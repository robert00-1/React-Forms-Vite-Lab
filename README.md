# Title 
Shopping List App

# Description
This is a simple react shopping list application that allows users to view a list of shopping items, filter items by category(produce,Dairy,Dessert),Search items by name using a controlled input, add new items to the list using form, toggledark/light mode.
# Tools
vite
react
node.js & npm
jest
css
javascript

# Setup
follow this instruction to access and run shopping list app

-clone the repository into your local machine terminal
git clone https://github.com/robert00-1/React-Forms-Vite-Lab.git

-navigate into the project folder
cd React-Forms-Vite-Lab
- install whole dependancies
npm install 
npm run dev
npm test
copy  andpaste the url that will appear after running npm run dev into your browser to open the shopping list app
